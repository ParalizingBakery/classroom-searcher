GC Class Searcher

An extension that lets users search classes in Google Classroom. Available on home page and archived page. Searching with teacher name is not supported if the user is a teacher.

Permissions

If you are on Firefox, the permission to edit Google Classroom is not granted automatically. This is required for the extension to function.
